# 누락값이란?


```python
from numpy import NaN, nan, NAN
```


```python
NaN == 0
```




    False




```python
NaN == False
```




    False




```python
NaN == ''
```




    False




```python
NaN == NaN
```




    False



비교할 값이 없는 것이니 상기의 사례에서 모두 False 출력

## 누락값 확인하는 방법: pd.isnull()


```python
import pandas as pd

pd.isnull(NaN)
```




    True




```python
pd.notnull(NaN)
```




    False




```python

```
